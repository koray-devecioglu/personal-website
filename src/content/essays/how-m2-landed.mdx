The second milestone locked in three things: a token layer in OKLCH,
three typefaces served from `/public/fonts`, and a handful of UI
primitives that the rest of the site can lean on.

<Callout tone="tip" title="Why OKLCH">
  Perceptual uniformity — a 10% lightness change looks like a 10% lightness
  change, which means I can derive the accent tint and the accent-strong shade
  from one hue without fighting the color wheel.
</Callout>

The next milestone is this — a typed content engine, a CLI for new
posts, and enough validation that I can't publish a malformed file
even if I try.
