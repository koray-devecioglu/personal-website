A working list of what I use. Nothing here is sponsored; everything
here is paid for or self-hosted. Tools only stay on the list if I'd
buy them again.

## Editor & shell

- Cursor (on top of VS Code) for day-to-day engineering.
- Zsh + [oh-my-zsh](https://ohmyz.sh) with a small custom prompt.
- Tmux for long-running sessions on remote boxes.
- Ghostty as the terminal, after a long stint in iTerm2.

## Writing

- Markdown + MDX in VS Code. No CMS, no WYSIWYG. The repo is the CMS.
- Obsidian for private notes — anything that might graduate to a post
  starts there first.

## Hardware

- 14" MacBook Pro (M-series, 32 GB) as the primary machine.
- 27" 4K external panel, mechanical keyboard, a trackball that
  divides every visitor into two camps.
- Noise-cancelling headphones I don't bother naming because the
  recommendation changes with the refresh cycle.

## Browser

- Chromium-based daily driver (DuckDuckGo as the default engine).
- Firefox for anything that needs real containers.
- Safari for reading in Reader Mode without extensions looking on.

## Services I pay for

- A couple of cloud VPSes I use to host small things.
- A password manager (because the alternative is unprofessional).
- A backup target I will absolutely recommend by name once I've
  trusted it through one real restore.
